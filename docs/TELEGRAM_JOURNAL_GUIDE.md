# 프리즘 어드바이저 기능 소개: /journal & /evaluate 활용 가이드

> 텔레그램 채널 공유용 안내문

---

## 📝 /journal 이란?

**투자 생각을 기록하는 나만의 투자 일기**입니다.

기록한 내용은 봇이 **기억**하고, 나중에 `/evaluate` 상담이나 `/journal` 대화에서 맥락을 파악하는 데 활용됩니다.

---

## ✅ /journal 올바른 활용법

### 1️⃣ 매매 결정 기록하기

```
/journal
⚠️ 포트폴리오 조정: 삼성전자(005930)
손절가: 54,000원으로 상향
이유: 강세장에서 수익 일부 확보
```

### 2️⃣ 시장 관점 기록하기

```
/journal
AI 반도체 섹터 과열 우려됨.
HBM 관련주 비중 축소 고려 중.
```

### 3️⃣ 종목 관찰 기록하기

```
/journal
AAPL 200달러 돌파 시 매수 고려.
실적 발표 전까지 관망.
```

### 💡 Tip: 종목코드나 종목명을 함께 적으면 더 잘 기억해요!

- ✅ "삼성전자 손절가 조정"
- ✅ "005930 목표가 상향"
- ✅ "AAPL 분할 매수 계획"

---

## 🔄 /journal + /evaluate 콜라보 활용

### Step 1: 매수 전 생각 기록

```
/journal
삼성전자 52,000원에 매수 진입.
목표가 60,000원, 손절가 48,000원.
HBM 수요 증가 기대.
```

### Step 2: 보유 중 평가 요청

```
/evaluate
삼성전자
52000
2주
```

→ 봇이 **이전에 기록한 매수 근거와 목표가**를 참고하여 더 맥락 있는 조언을 제공합니다!

### Step 3: 매도 후 복기

```
/journal
삼성전자 58,000원에 익절.
목표가 도달 전 매도했지만 +11% 수익 만족.
다음엔 분할 매도 고려.
```

## 💬 /journal 답장 대화 기능

저널을 저장한 후, **봇의 답장에 다시 답장**하면 AI와 대화를 이어갈 수 있어요!

```
You: /journal
     삼성전자 손절가 조정했어

Bot: ✅ 저장되었습니다!
     💡 답장으로 대화를 이어가세요!

You: [답장] 지금 상황에서 어떻게 생각해?

Bot: 📊 삼성전자에 대한 기존 기록을 확인했어요.
     이전에 목표가 60,000원을 설정하셨는데...
```

## 📚 /memories - 내 기억 확인하기

저장된 기록을 확인하고 싶다면:

```
/memories
```

→ 종목별 기록 개수, 최근 저널 내역을 한눈에 볼 수 있어요!

## 🎯 요약

| 명령어 | 용도 | 활용 |
|--------|------|------|
| `/journal` | 투자 생각 기록 | 매매 결정, 시장 관점, 관찰 |
| `/evaluate` | 보유 종목 상담 | 과거 기록 참고한 맞춤 조언 |
| `/memories` | 기록 확인 | 저장된 기억 조회 |

**기록할수록 똑똑해지는 프리즘 어드바이저!** 🧠✨
