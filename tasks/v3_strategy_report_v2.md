## 3. "비중"은 어떻게 정하나 (risk%의 뜻)

이 시스템은 비중을 고정하지 않고 **"한 번 손절당했을 때 계좌가 깎이는 비율"**(risk)을 고정한다.

```
포지션 크기(명목) = 계좌 × risk% ÷ 손절폭(~1.4%)
```

예시: 계좌 1,000만원, risk 3%일 때
- 한 번 손절 = **-30만원 (계좌의 -3%)** ← 이게 고정되는 것
- 포지션 명목가 ≈ 30만원 ÷ 1.4% ≈ **2,100만원 (계좌의 약 2.1배 베팅)**
- 10배 레버리지면 증거금 210만원만 잠김 (계좌의 21%) — 나머지는 현금 대기
- 6연패(최악 실측) 시: 계좌 약 -17%

즉 "비중 몇 %?"의 답: **명목 기준 계좌의 1.5~3배를 들어가되, 손절이 1.4%로
짧아서 실제 1회 리스크는 계좌의 2~4%.** 레버리지는 증거금 절약 수단일 뿐.
